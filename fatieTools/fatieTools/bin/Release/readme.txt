软件使用说明：
软件有3个功能
1 登录 

登录功能需要选择账号存储的文件
要求，utf-8编码，格式为：账号----密码 一行一个账号
例：
abmy----abmy
心爷----xinye

在登录完成或者登录过程中可以按保存，会把登录以后的cookie存放在软件目录下的cookie文件夹里面，会生成相应的文件

2 签到
点击导入以后会自动导入cookie文件夹里面的文件，然后点开始签到就可以了


3 回帖
账号会自动导入，需要召唤的人需要导入文件夹
要求： utf-8编码，格式为 ：用户名 一行一个用户名
例
abmy
心爷

然后输出内容，记得最后多出回车，然后点击开始召唤就可以了

发帖方式使用的是pc端的post方式，换行方式为[br]，空格为&nbsp;

文件解释
result/result.txt 是召唤结果名单，只要是被召唤过的人都会存在这里面，避免重复召唤
cookie里面存放的是cookie 名单列表和每个人的cookie
